Лабораторна робота №5 Тема: Узагальнені типи (Generics), колекції і LINQ, обробка винятків
Мета: Навчитися створювати узагальнені класи та методи, працювати з колекціями та LINQ, а також коректно валідувати дані й обробляти винятки.
Опис проєкту: Даний проєкт демонструє роботу з узагальненими типами, колекціями та запитами LINQ у мові програмування C#. У прикладі використовується узагальнений клас Repository, який дозволяє зберігати та обробляти будь-які типи об’єктів. Також реалізовано власний виняток InvalidGradeException для перевірки коректності оцінки студента. Програма демонструє додавання об’єктів до колекції, фільтрацію студентів за допомогою LINQ, валідацію даних і обробку винятків.
Структура проєкту: Lab5/ Program.cs – головна програма Repository.cs – узагальнений клас Repository Student.cs – клас моделі Student InvalidGradeException.cs – власний клас винятку README.md – опис проєкту
Інструкція з запуску:
1. Завантажити або клонувати проєкт.
2. Відкрити у Visual Studio або Visual Studio Code.
3. Переконатися, що встановлено .NET 7.0 або новішу версію.
4. Запустити файл Program.cs.
5. Переглянути результати роботи програми у консолі.
Особливості реалізації:
* Використовується узагальнений клас Repository для зберігання об’єктів.
* Реалізовано методи додавання, отримання та пошуку елементів.
* Застосовано LINQ для фільтрації студентів за оцінками.
* Додано обробку стандартних і власних винятків через try-catch-finally.
* Налаштовано підтримку українських символів у консолі.
Приклад результату роботи програми: Студенти з оцінкою більше 80: Іван, Age: 20, Grade: 85 Оля, Age: 22, Grade: 92 Анна, Age: 21, Grade: 88 Власний виняток: Оцінка не може бути більше 100 Програма завершила виконання.
